Voici les étapes à suivre :
     Le groupe se partage la phrase. Chacun doit avoir au moins un mot à écrire. 
     Vous devez ajouter ce mot dans le fichier texte au niveau de votre branche personnelle.
     Faire remonter les modifications pour que la phrase soit complétée dans le main.
